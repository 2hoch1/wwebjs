## About

This repository contains multiple packages with separate [releases][releases]. You can find whatsapp-web.js in our [other repository][other-repo], which allows you to interact with WhatsApp Web and build applications on top of it.

## Apps

- `website` ([source][source-website]) • The Guide and Documentation for whatsapp-web.js

## Packages

- `whatsapp-web.js` ([source][source-wwebjs]) • A powerful Node.js module for build WhatsApp applications
- `create-wwebjs-app` ([source][source-create-wwebjs-app]) • A CLI tool to quickly create a whatsapp-web.js project
- `@wwebjs/formatters` ([source][source-wwebjs-formatters]) • A collection of utility functions for formatting WhatsApp messages
- `@wwebjs/cli` ([source][source-wwebjs-cli]) • A CLI tool with several useful features to make your life easier
