## Introduction

**whatsapp-web.js** <span className='text-fd-muted-foreground text-sm'>(WWebJS)</span> is a Node.js library for interacting with WhatsApp Web, using Puppeteer to run a managed browser and reduce blocking risk.

WWebJS has different parts:

<Cards>

<Card icon={<CpuIcon className="text-purple-300" />} title='whatsapp-web.js' href='https://github.com/wwebjs/whatsapp-web.js'>

Handles client, messages, chats, and communication with WhatsApp Web.

</Card>

<Card icon={<PanelsTopLeftIcon className="text-blue-300" />} title='WWebJS Core'  href='https://github.com/wwebjs/wwebjs/tree/main/packages/core'>

Utilities for sessions, authentication, and client lifecycle.

</Card>

<Card icon={<DatabaseIcon />} title='WWebJS Formatters' href='https://github.com/wwebjs/wwebjs/tree/main/packages/formatters'>

Tools for formatting messages and media.

</Card>

<Card icon={<TerminalIcon />} title='WWebJS CLI' href='https://github.com/wwebjs/wwebjs/tree/main/packages/cli'>

CLI tools for setting up your app.

</Card>

</Cards>

## Automatic Installation

A minimum version of Node.js 18 is required.

```npm
npm create wwebjs-app
```

It will ask you the built-in steps:

<Steps>
  <Step>
    **Project name**

    Choose the name for your new project.

  </Step>

  <Step>
    **Select a template:**

    - JavaScript
    - TypeScript

  </Step>

  <Step>
    **Use a `src/` directory?**

    Decide whether your files should be placed in a dedicated `src/` folder.

  </Step>

  <Step>
    **Auth strategy:**

    - NoAuth (no session persistence)
    - LocalAuth (local session storage)
    - RemoteAuth (database session storage)

  </Step>

  <Step>
    **Package manager**

    Choose your preferred package manager
    - `npm`
    - `pnpm`
    - `yarn`

  </Step>

  <Step>
    **Install dependencies automatically?**

    Confirm whether the CLI should install dependencies right away.

  </Step>
</Steps>
