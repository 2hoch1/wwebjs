## Functions

### `bold(text)`

Wraps text in WhatsApp bold formatting (`*text*`).

| Parameter | Type | Description |
|-----------|------|-------------|
| `text` | `string` | The text to format |

### `italic(text)`

Wraps text in WhatsApp italic formatting (`_text_`).

| Parameter | Type | Description |
|-----------|------|-------------|
| `text` | `string` | The text to format |

### `strikethrough(text)`

Wraps text in WhatsApp strikethrough formatting (`~text~`).

| Parameter | Type | Description |
|-----------|------|-------------|
| `text` | `string` | The text to format |

### `monospace(text)`

Wraps text in WhatsApp monospace formatting (` ```text``` `).

| Parameter | Type | Description |
|-----------|------|-------------|
| `text` | `string` | The text to format |

### `code(text)`

Wraps text in inline code formatting (`` `text` ``).

| Parameter | Type | Description |
|-----------|------|-------------|
| `text` | `string` | The text to format |
